### 1. Connect Zapier
Navigate to the Integrations tab in the Surface App and connect Zapier. Make sure you have a Zapier account.
<img className="block" src="/images/zapier/zapier_1.png" alt="Hero Light" />

Accept the invitation to get started.
<img className="block" src="/images/zapier/zapier_2.png" alt="Hero Light" />

### 2. Create a new Zap and search for the Surface Forms 
Select the latest Surfae Forms version that appears.
<img className="block" src="/images/zapier/zapier_4.png" alt="Hero Light" />

Select `New Response` as the Event option.
<img className="block" src="/images/zapier/zapier_3.png" alt="Hero Light" />

### 3. In the Account tab, connect a new account.
You should see the following screen appear:
<img className="block" src="/images/zapier/zapier_9.png" alt="Hero Light" />

### 4. Create API Key
To get the API key, you need to first create a new production key from the Surface app.

Navigate to Settings > API Keys in the Surface app and add a new Production Key (use any API Key label).

Copy the API Key that appears and paste it into the API Key field from **Step 3**.
<img className="block" src="/images/zapier/zapier_10.png" alt="Hero Light" />

### 5. Add the Survey ID to Zapier
To find the Survey ID, navigate to the form in the Surface app and click on the share icon (to the left of the Edit button). The value that appears after `https://app.withsurface.com/s/...` is the Survey ID.
<img className="block" src="/images/zapier/zapier_7.png" alt="Hero Light" />

Paste this value into the Survey ID field in the Zapier Trigger tab. 

For "Process Partial Responses", select either `Yes` or `No` depending on if you would like to trigger the Zap even on partial form responses.
<img className="block" src="/images/zapier/zapier_8.png" alt="Hero Light" />


### 6. Test the Trigger to ensure that you are able to view Surface form responses
<img className="block" src="/images/zapier/zapier_11.png" alt="Hero Light" />

### 7. Finally, connect an action!
<img className="block" src="/images/zapier/zapier_12.png" alt="Hero Light" />
