Surface Forms can be integrated with Zapier to automate workflows with thousands of apps. Connect your forms to your favorite tools without writing any code.

## Prerequisites

Before connecting to Zapier, you'll need:
- A Surface Forms account
- A Zapier account
- API key from Surface Forms settings

## Integration Steps

<Steps>
  1. Navigate to the **Integrations** panel from the sidebar
  2. Find the Zapier card and click **Connect**
  3. Accept the invitation to use Surface Forms on Zapier
  4. Create a new Zap or use existing one
  5. Add Surface Forms as your trigger app
</Steps>

## Setting up API Authentication

<Steps>
  1. Go to Surface Forms Settings tab
  2. Generate a new API key
  3. Copy the API key
  4. In Zapier, choose Surface Forms as trigger
  5. Paste your API key when prompted
  6. Click "Continue" to validate connection
</Steps>

<Note>
  Keep your API key secure. If compromised, generate a new one from the Settings tab.
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Connection Issues">
    1. Verify API key is correct
    2. Check if API key is still valid
    3. Ensure you have accepted the Zapier invitation
  </Accordion>
  
  <Accordion title="Missing Triggers">
    1. Verify form is published
    2. Check if Zap is turned on
    3. Confirm trigger conditions are met
  </Accordion>
</AccordionGroup>
