Workflows allow you to automate actions based on form interactions and lead activities. Create sophisticated automation flows that trigger when users interact with your forms.

## How Workflows Work

<Steps>
  1. Choose a source (e.g., Surface Forms)
  2. Select specific form triggers
  3. Define when workflow should run
  4. Add actions to be executed
  5. Configure action settings
</Steps>

## Workflow Triggers

Workflows can be triggered at different stages of form interaction:

<CardGroup cols={1}>
  <Card title="Form Submission" icon="paper-plane">
    <ol>
      <li>1. Triggers after user completes the form</li>
      <li>2. All form data available</li>
      <li>3. Best for final actions</li>
    </ol>
  </Card>
  
  <Card title="In Progress" icon="spinner">
    <ol>
      <li>1. Triggers after specific step completion</li>
      <li>2. Partial form data available</li>
      <li>3. Good for progressive actions</li>
    </ol>
  </Card>
  
  <Card title="Dropped Off" icon="user-xmark">
    <ol>
      <li>1. Triggers when user abandons form</li>
      <li>2. Captures partial submissions</li>
      <li>3. Useful for recovery campaigns</li>
    </ol>
  </Card>
</CardGroup>

## Creating a Workflow

<Steps>
  1. Click "Create workflow" button
  2. Select "Surface forms" as the source
  3. Choose your specific form
  4. Select trigger event:
     - Form Submission
     - In Progress (Step0)
     - User Dropped Off
  5. Add desired actions
  6. Configure action settings
  7. Activate workflow
</Steps>
